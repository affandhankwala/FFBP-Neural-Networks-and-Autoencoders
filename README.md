## Introduction
This project is in accordance with Johns Hopkins Introduction to Machine Learning course number 605.649. 

## Problem Statement
This problem consists of designing, developing, training, and testing a feed forward neurla network with backpropagation. This backpropagation is vital for adjustments of edge weights. We also implement an autoencoder and analyze its benefits to the classification and regression tasks.
